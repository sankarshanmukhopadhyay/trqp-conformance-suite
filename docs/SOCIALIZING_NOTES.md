# Socializing Notes

## What this repo is

- An independent TRQP conformance harness
- Profile-driven requirements and executable tests
- Evidence bundles designed for independent verification

## What this repo is not

- An official Trust over IP Foundation artifact
- A certification authority
- A replacement for the TRQP specification

## Suggested phrasing

- "A reference conformance harness to make TRQP testable and auditable"
- "Profiles let ecosystems adopt higher assurance without imposing it on everyone"

Avoid "official" or "certification" language.
