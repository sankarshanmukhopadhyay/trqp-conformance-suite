# v0.4.4 — Consolidated conformance baseline (GRID-aligned)

This release establishes v0.4.4 as the current 0.4.x baseline while adding optional, non-breaking support for GRID-style directory artifacts.

## Added
- Optional JSON Schemas for:
  - Registrar listing
  - GRID status feed
- `docs/GRID_SUPPORT.md` clarifying what the baseline runner does and does not enforce.

## Notes
- This suite validates artifact **shape** only; deployments MUST define cryptographic proof formats and verification rules.
