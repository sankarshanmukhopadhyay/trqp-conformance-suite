# TRQP Conformance Suite v1.5.0 Release Notes

## Summary

v1.5.0 is the CTS portion of the Operational Trust Stack Maturity Release. It positions the suite as a stable verification engine with explicit release thresholds, validation evidence, and compatibility expectations.

## Highlights

- Added release governance for patch, minor, maturity, and no-release changes.
- Added change-intake criteria for requirement, profile, evidence, validation, and compatibility impact.
- Added release validation documentation for the coordinated Hub v1.8.0 / CTS v1.5.0 / TSPP v0.13.0 tuple.
- Refreshed README and roadmap language around high-value, evidence-producing CTS commits.
- Updated GitHub PR and release templates to require maturity-gate evidence.

## Compatibility

| Repository | Version |
|---|---:|
| TRQP Conformance Suite | v1.5.0 |
| TRQP-TSPP | v0.13.0 |
| TRQP Assurance Hub | v1.8.0 |
