# Scenario Profiles

Scenario profiles bind protocol conformance testing to business decisions. They are designed for adoption workflows where an operator needs to select a profile by use case rather than by implementation detail.

| Profile | Decision supported | Baseline AL |
|---|---|---|
| `scenario_ecosystem_onboarding` | Admit a registry into an ecosystem. | AL2 |
| `scenario_public_directory` | Rely on an authoritative public directory. | AL3 |
| `scenario_supply_chain_registry` | Accept supply-chain participant registry claims. | AL2 |
| `scenario_agent_service_registry` | Accept agent service registry metadata. | AL3 |
