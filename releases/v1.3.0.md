# v1.3.0 — Scenario Profiles and Interop Evidence Matrix

## Release purpose

This release makes CTS evidence easier to use in adoption decisions. It adds scenario profiles for ecosystem onboarding, public directories, supply-chain registries, and AI agent service registries, plus an interop evidence matrix generator.

## Highlights

- New scenario profile YAML files.
- New `scripts/generate_interop_matrix.py` tool.
- Scenario run examples and matrix samples.
- Relying-party evidence documentation.
- Cross-repo references refreshed for Hub v1.6.0 and TSPP v0.11.0.
